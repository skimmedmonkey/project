Create a D&D Character

Use the below form to create a character for Dungeons and Dragons. When setting your stats such as strength or dexterity, remember to keep the stat at or below 20. Type in the equipment you want the character to start with, you can be creative since you are the Dungeon Master. List out the attacks your character has, like shortsword 1d6+2 so it's easy to know how many dice you need to roll, as well as their modifier. The spells are optional, but if you'd like, you can include any number of spells like Magelight or Acid Rain, as a couple examples. When you're finished, click Generate Character and see it displayed in the web app for easy use.
